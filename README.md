# Habit Tracker

A React habit tracking app built from scratch.

## Features
- Add and delete habits
- Mark habits as done each day
- 🔥 Streak counter
- Weekly dot view showing last 7 days
- Persists data with localStorage
- Mobile responsive

## Live Demo
https://habit-tracker-ten-steel.vercel.app/

## Built With
- React + Vite
- CSS
- localStorage
